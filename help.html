The objective for this class is offer the most simple way to create a login system in your site.
If you dont understand how OOP works, you only need use the example and this help for you have one login system work in you site.
----
Login:
For login you only need create the form with username and password inputs. Need to create a hidden field with "dologin" name.
<input type="text" name="username" />
<input type="text" name="password" />
<input type="hidden" name="dologin" value="dologin" />
Logout:
For logout you only need create the form with input hidden with name "dologout".
<input type="hidden" name="dologout" value="dologout" >
Redirect:
For redirect you need create a input hidden with name "redirect" and value for url you want redirect the user after login or logout.
<input type="hidden" name="redirect" value="#loged" >
Place Messages:
<div class="message"><?php echo $login->message; ?></div>
Init Class HandlerForm:
$login = new HandlerForm();
Init Class ValidateUser:
$user = new ValidateUser();
Get Username:
$user->getUserName();
Get UserID:
$user->getUserId();
Login with email:
If you dont allow user login with email, change EMAIL_LOGIN to false.
Default is true.
define("EMAIL_LOGIN", true);
